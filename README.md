# Calculator

A simple calculator built with HTML, CSS and JavaScript.

## Overview

A web interface to perform basic math operations: addition, subtraction, multiplication and division.

## Technologies used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NorihyDev/Mini-calculator.git
   ```
2. Open the `index.html` file in your browser.

## Features

- Addition, subtraction, multiplication, division
- Simple and intuitive interface
